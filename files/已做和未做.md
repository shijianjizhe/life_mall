# 已做和未做状态（2026-07-09 更新）

## 本轮已补齐

商品图片体系  
136 个种子商品已补齐 `mainImageUrl` 和 2 张 `galleryImages`，JPG 图片资产位于 `public/products/`。商品列表、详情轮播、购物车、订单、房间、海报、100 万专题均通过 `ProductVisual` 优先展示商品图，图片失败时保留 emoji fallback。

加购飞入购物车动效  
商品卡和商品详情页加购会触发缩略图飞入购物车入口的全局动画，购物车目标支持顶部和底部导航定位。

移动端震动反馈  
新增 `safeVibrate`，加购、房间摆放、吸附、删除、签到、支付完成等关键节点有轻触感反馈；不支持震动的浏览器会静默降级。

虚拟房间双指缩放  
房间物品支持移动端两指缩放，同时保留桌面滚轮缩放和按钮缩放。

虚拟房间磁吸辅助线  
拖拽时支持画布中心线、其他物品中心线吸附，命中时显示横/竖辅助线。

签到快乐币数字滚动动画  
签到成功后快乐币数字通过 `AnimatedNumber` 从旧值滚动到新值。

结算支付动画更精细化  
假支付 loading 已拆成阶段式进度：校验快乐、联系平行宇宙仓库、假装扣款、生成永不送达面单，并防止重复提交。

分享降级体验  
海报页和房间页统一分享逻辑：支持系统文件分享；不支持时自动下载图片，并提示可长按/从下载中发送，分享文案可复制时会复制。

AI 报告 4 屏仪式感  
AI 人格报告已拆成封面、标题揭晓、数据支撑、人格点评四屏。

备份导入校验  
导入校验已加严：校验日期、枚举、数量范围、金额范围、订单 item 和核心 AI setting。

分类页主题差异  
四个分类页已增加独立页面背景、商品卡样式、价格色和描边风格。

海报模板  
海报使用商品图拼贴，并补充空状态分享画面。

PWA 构建验证  
`npm run build` 已通过，PWA manifest、service worker 和 workbox 文件已生成。

README / 计划文档收口  
README 已从默认 Vite 模板说明替换为 Life Mall 项目说明，PLAN 已更新完成状态。

## 已有但仍可继续打磨

商品图片美术质量  
当前图片为本地生成的统一扁平 JPG 资产，已满足功能和视觉占位要求；后续可替换为更精致的 AI 插画或人工设计图，只需覆盖 `public/products/` 对应文件。

商品详情 AI 客服底部抽屉  
目前商品详情仍是跳转全屏 AI 聊天页。文档允许“抽屉或全屏”，所以不算硬缺；如果追求更贴近商品详情沉浸感，可以再补一个简易底部抽屉。

深色模式  
开关和基础变量覆盖已有，部分特殊渐变/主题卡片后续仍可继续精调。

100 万专题  
预算、进度、超支、完美花光彩蛋、海报都有；“财富自由挑战达成”目前已有文案和庆祝反馈，后续还可做更强称号视觉。

PWA 离线体验  
构建产物已生成 service worker；真实断网离线缓存策略可在浏览器里继续做专项验证。
